Redovisning kmom01
=========================

Detta innehåll är skrivet i markdown och du hittar innehållet i filen `content/redovisning/01_kmom01.md`.

Fråga 1: Jag har aldrig använt mig av ramverk förut. Jag har designat hemsidan förut men har mycket att lära mig. Jag brukar gå för en enkel design när jag skapar en hemsida.

Fråga 2: Det var svårt från början att lösa allt med labbmiljön. Från början tyckte jag att det var oklart hur man skulle ladda ner allt och få se det att fungerar. Det tog lite tid att hitta alla koder som man ska använda för att ladda ner labbmiljön med det löste sig. Jag fick ett stort problem när jag skulle fixa min github, så blev tvungen att ladda ner allt om igen och börja från nytt.

Fråga 3:Jag trodde inte att man skulle koda hemsidor jag trodde man skulle designa en hemsida genom ett program eller så skulle man bara använda sig av css och koda hemsidor som vi gör just nu. Jag trodde att vi skulle fixa designen på hemsidan inte att lära oss ett nytt språk som vi kommer att göra i kmom02.

Fråga 4: Jag är ganska van med att koda css och tycker att det är enkelt och roligt. Det gick ganska bra med att använda css men det enda jag hade svårt var att försöka hitta vad klassen heter.

Fråga 5: I den hära uppgiften lärde jag mig lite mer om css och hur jag kan strukturera min css kod så att det är lätt att hitta allt.
