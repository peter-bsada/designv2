Redovisning kmom05
=========================

Fråga 1: Erfarenheterna jag fick genom att undersöka webbplatsen var hur man kan förbättra laddningstiden. Jag kom fram till att man kan ta bort onödiga css som inte är använt.

Fråga 2: Jag har aldrig använt mig utav Cimage förut. Men jag tyckte att det var ganska enkelt att förstå hur man arbetar med det, speciellt efter jag läste verktyg som Mikael visade. Jag brukar inte använda någon bild verktyg, för att jag inte har behövt att ändra nått eller fixa något till bilden.

Fråga 3: Jag hade det svårt från början att förstå vad man skulle göra. Men efter att ha läst flera gånger började förstå koden. Git länken som finns i dbwebb som Mikael visade hur man löste @desinax hjälpte mig ganska mycket att förstå, utan den tror jag inte att jag skulle klara mig.

Fråga 4: Jag visste inte att en bild kan ta så mycket plats i en sida, därför är det viktig att fixa bilden så det inte tar så mycket plats i hemsida. Det står större bild det är det stå längre är nedladdningstiden och det är samma med responsiva bilder.

Fråga 5: Jag lärde mig hur man arbetar med Cimage och att man ska tänka på att inte ha så stora bilder så att nedladdningstiden inte blir för länge. Jag också lärt mig hur man kan förbättra nedladdningstiden till en hemsida.
